# Pharmacy Management System

## Project Overview
The Pharmacy Management System is a comprehensive solution designed to streamline pharmacy operations. It facilitates the management of medications, prescriptions, sales, and inventory, ensuring efficient processes and improved customer service.

## Features
- User Authentication
- Prescription Management
- Inventory Tracking
- Sales Reporting
- User Roles and Permissions
- Search Functionality for Medications
- Responsive Design for Accessibility on Various Devices

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git
- Deployment: Heroku

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ashirushehu2015-stack/Pharmacy-management-system.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Pharmacy-management-system
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up environment variables as specified in `.env.example`.
5. Start the server:
   ```bash
   npm start
   ```

## Usage
- Open your browser and navigate to `http://localhost:3000` to access the application.
- Follow the on-screen instructions to log in or register an account.
- Utilize the navigation menu to access different sections of the application.
